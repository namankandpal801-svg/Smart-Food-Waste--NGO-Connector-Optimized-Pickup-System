## 🚀 Overview  
A platform that connects food donors with NGOs to reduce food waste and ensure surplus food reaches people in need.

## ✨ Features  
- 🔐 Donor & NGO Authentication  
- 🍱 Food Listing System  
- 🔄 Request Matching  
- 🚚 Pickup Scheduling  

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS  
- **Backend:** Python (Flask)  
- **Database:** SQLite / MySQL
